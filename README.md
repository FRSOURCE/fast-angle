# Fast angle

Simple online protractor - measure any angle directly on your image!

## Development

> Before stating development, make sure to install package dependencies by running `pnpm i`

```bash
# running dev app
pnpm dev

# production build
pnpm build

# production build preview
pnpm preview

# running native apps
pnpm preview:ios
pnpm preview:android

# unit tests
pnpm test

# typecheck
pnpm typecheck

# lint autofix
pnpm lint --fix
```
